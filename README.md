### Development server
Start express server [`http://localhost:3000/`](http://localhost:3000/))
 ```sh
$ npm run node-server
``` 
Build project (folder `/dist`)
 ```sh
$ npm run start-build
``` 
Start webpack-dev-server [`http://localhost:9000/`](http://localhost:9000/))
 ```sh
$ npm run start-dev
```
